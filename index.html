What Is Programming?

A beginners introduction to programming, various languages and their uses

A computer program is a set of instructions that a computer reads and executes in order to perform various tasks. Think of it like a complicated, precise recipe that will be cooked by a perfectionist chef. All of the instructions have to be perfect or else it could all fail. If the recipe is written properly, you’ll get an amazing dish at the end. Same thing with programming. Computers can’t interpret anything or think for themselves. They have to be told exactly what to do. If you give it the correct instructions, then you get the correct result. These programs can do anything from analyzing data, moving robots, to creating websites, or to making video games. These instructions are written in various coding languages, and I hope to introduce you to a few of them on this website. In fact, this website was coded by me for a school project. This website you see before you, your web browser, your device’s operating system, all of that works and exists thanks to programming. Hopefully I’ll be able to help teach you a few things about this subject, as well as some information about a couple of languages. And if you're already an expert, then maybe you’ll enjoy judging my interpretations of programming. Or maybe you want a refresh. But anyways, let’s get right into it.

Programming or Coding: What's the difference?

You’ve probably heard of both programming and coding and thought, “what’s the difference? Is there one?” Well there sort of is, and I’ll explain it, but generally you can use either word and still get the same message across. Simply put, coding is the writing of instructions that both you and the computer can read. Programming is the guide the computer follows in order to execute the code properly. Going back to the cooking example, we can compare coding to the recipe. You wrote the recipe, and the chef understands it and follows it to make an amazing dish. You wrote the code, and the computer understands it and does whatever you’ve asked. Meanwhile, programming can be compared to the tasks. Tasks involve anything the recipe asks you to do. This includes things like boiling, whisking, cracking eggs or kneading dough. Obviously a chef already knows how to do these things because they’ve been taught how. Programming teaches the computer how to execute the instructions given by the code. Hopefully you were able to understand that, and if you didn’t, that’s ok. It took me a while to understand the difference.

Code

Code will consist of precise instructions that the computer understands. In the recipe example it might look like “blend: basil (50 grams) with: olive oil (25 grams) - time: 34.2 seconds.” It has to be exact and properly formatted, so you can’t say something like “blend about 20-30 basil leaves with a medium-large splash of olive oil for around 30 seconds.” These instructions are too unclear for our chef to understand. Just like the chef, the computer can’t interpret what you want, it has to be told exactly what to do.

Languages

There are tons of different coding languages and they all have different purposes and specialties. Some are better at making websites, while others are better at telling robots where to go. As for how many languages there are, no one knows for sure. Some think that there are 250-2,500 official languages. However, anyone can create a programming language. Keeping this in mind, some people think that there are around 8,945 languages out there. In the end though, all of the languages just have different ways for instructions to be written, and some languages have more or less instructions than others.
